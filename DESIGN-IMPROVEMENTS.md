## 🎨 Guía para Buscar/Tomar Fotos de Reemplazo

### Características Esenciales
- **Iluminación**: Natural, cálida, suave
- **Colores**: Verde bosque, sage, ivory, linen (paleta de marca)
- **Estilo**: Artesanal, auténtico, no muy pulido
- **Mood**: Cálido, familiar, hecho a mano

### Especificaciones Técnicas
- **Formato**: JPG para fotos, PNG para gráficos
- **Tamaño**: Mínimo 1920px en el lado más largo
- **Peso**: Optimizar a < 500KB
- **Aspect Ratio**: 
  - Hero: 3:2 o 4:3
  - Proceso: 1:1 (cuadrado)
  - Story: 3:4 (vertical)

## 💡 Tips para Fotografía DIY

Si quieres tomar fotos propias sin fotógrafo profesional:

### Equipo Básico
- Smartphone moderno (iPhone/Android reciente)
- Luz natural (ventana grande)
- Fondo simple (pared blanca, tela neutra)

### Configuración
- **Hora**: Golden hour (amanecer/atardecer)
- **Luz**: Indirecta, desde ventana
- **Fondo**: Simple, sin distracciones
- **Composición**: Regla de tercios

### Edición Básica
- Ajustar brillo y contraste
- Corregir color (cálido, no frío)
- Crop para composición
- Comprimir para web

Cuando tengas nuevas imágenes:

1. **Optimizar**
   ```bash

   ```

2. **Renombrar**
   ```bash

   hero-lifestyle.png
   mother-daughter.png
   hands-crochet.png
   yarn-materials.png
   ```

3. **Copiar**
   ```bash
   wsl -d Ubuntu-22.04 -- zsh -c "cd /mnt/c/Users/p_bre/alma-and-hilo-website && cp nueva-imagen.png public/images/hero-lifestyle.png"
   ```

4. **Commit y Deploy**
   ```bash
   wsl -d Ubuntu-22.04 -- zsh -c "cd /mnt/c/Users/p_bre/alma-and-hilo-website && git add -A && git commit -m 'feat: actualizar imágenes hero/proceso con fotos reales' && git push && source ~/.zshrc && vercel --prod"
   ```
